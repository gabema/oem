Usage

The Responsifier's queries only fire when component initilization (which is automatic) is complete or on a page resize.

There are two elements to consider: 1) the element that is being watched (aka the container) and 2) the element that the classes are being applied to (has to be an Oem Component).

Note: Components can be their own "containers" if the classes you add don't affect the same dimension you're using to apply those same classes!

Attributes

attr desc
data-oem-component Oem component classes are to be applied to
data-oem-container Component/Element to watch. If not defined, the component will be considered it's own container. Can be an Oem Component id or basic DOM id here.
data-oem-responsive-class Class to be applied
data-oem-dimension Dimension to watch ("width" or "height")
data-oem-min Minimum dimension
data-oem-max Maximum dimension (use "*" for no max)

Html


<div id="exampleContainer">
    <button 
        data-oem="Button" 
        data-oem-id="ExampleComponent">
        Normal
    </button>
</div>

<div 
    data-oem="Responsifier" 
    data-oem-id="responsifierExample"
    data-oem-component="ExampleComponent"
    data-oem-container="exampleContainer"
    data-oem-responsive-class="--hollow"
    data-oem-dimension="width"
    data-oem-min="0"
    data-oem-max="360"></div>
